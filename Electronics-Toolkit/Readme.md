**Electronics Toolkit**

A collection of Python programs related to basic electrical and electronic calculations.

__About__

This project contains multiple calculators that help solve common electronics and electricity problems. The programs were built while learning Python and electrical concepts.

__Included Tools__

_1. Ohm's Law Calculator_

Calculates:

- Voltage (V)
- Current (I)
- Resistance (R)

Using Ohm's Law:

- V = I × R
- I = V / R
- R = V / I

---

*2. Resistance, Voltage Drop and Current Calculator*

Features:

_Series Circuit_

- Total Resistance
- Voltage Drop across each component

_Parallel Circuit_

- Total Resistance
- Total Current in the circuit
- Current through each branch

---

_3. Power, Energy and Electricity Bill Calculator_

Calculates:

*Power*

Using:

- P = V × I
- P = I²R
- P = V²/R

*Energy / Work Done*

Using:

- E = P × t

Outputs:

- Energy in Joules (J)
- Energy in Watt-hours (Wh)

*Electricity Bill*

Calculates electricity cost using:

- Units Consumed (kWh)
- Cost per Unit (₹)

---
